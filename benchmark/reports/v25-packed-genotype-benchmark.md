# VariantFlow v2.5 Packed Genotype Benchmark

## Correctness Gate

All rows must pass VCFtools parity before any claim update. The supported replacement scope remains diploid biallelic genotype data.

## Caveat

This report is not a broad VCFtools replacement claim. It supports scoped claims for the measured 100k and 1M 1000 Genomes / IGSR diploid biallelic human cohort rows only. The 1M evidence completed all workflow outputs on 2026-05-09; the original Docker run exited after the old parity normalizer was killed on the 15M-row LD output, then `benchmark/check_vcftools_parity.py` was fixed to use disk-backed sorted LD group comparison and the existing 1M outputs passed normalized parity.
